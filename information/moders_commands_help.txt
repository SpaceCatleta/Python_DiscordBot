====================================
-mod - группа модерских комманд
===ограничение: только модераторы===
    moot [время сек.] [линк] - выдаёт плашку на временный мут в текстовых чатах
    voice [время сек.] [линк] - выдаёт плашку на временный мут в голосовых чатах
    banfunc [время сек.] [линк] - выдаёт плашку на временный бан функций бота
    clearmes [кол-во сообщений] [-exp] [линк] - удаляет последние сообщения
        [кол-во] - по уолчанию 10
        [-exp] - снять опыт за эти сообщения
        [линк] - с этим параметром удаляются сообщения только этого пользователя

====================================
-set - группа для изменения натроек
===ограничение: только модераторы===
    show - показывает текущие настройки

    general:
    time_update_delay [начение] - период обновления системной информации
    max_links [начение] - максимальное кол-во линков
    timeout [начение] - время до таймаута
    timeout_limit [начение] - макс. кол-во таймаутов
    bomb_mes_time [начение] - время системных сообщений

    roles:
    base_role [линк/название] - плашка поумолчанию
    ban_functions [линк/название] - плашка для бана функций бота
    moot [линк/название] - плашка для мута в тексовых чатах
    vc_moot [линк/название] - плашка для мута в голосовых каналах

====================================
-sys - группа системных комманд
===ограничение: только модераторы===
    sys_recalc_all - перерассчитывает статистику по сообщениям на сервере
    fixstats - перерассчитывает опыт и перепроверяет имена всех пользователей в БД
    off - выключение бота
