**-guild** - раздел команд настроек сервера
===ограничение: только администраторы===
**-guild guilds** - перечислить все сервера, где подключён бот
**-guild show** - вывести все настройки сервера
    осторожно - id сервера так же выводится
**-guild spam** - добавить(удалить)) канал (в котором вызвана команда) в список спам-каналов
**-guild set [название настройки] [значение]** - изменить одну из настроек
    узнать название настройки в -guild show
    роль устанавливается по линку или названию
    канал устанавливается по линку
    время устанавливается в секундах
**-guild update_levels** - проверяет соответствие exp уровню и ролей уровню всех пользователей на сервере
    при наличии у пользователя роли не соответствущей его уровню - роль забирается
    на выходе пользователю выдаётся одна роль в соответствии с его уровнем
**-guild level set [уровень] [роль]** - установить роль за уровень
**-guild level delete [уровень]** - удалить роль из списка
    роль устанавливается по линку или названию
    при удалении роли из списка - у пользователей роль не отнимается
    для удаления роли у всех - присвойте её на недостижимый уровень и вызовите -guild update_levels
