 версия бота - v2.4.0
 [ ] - необязательный параметр
-помощь - вызывает список команд
-tm - тестовоая команда
-bomb [линк] - повторяет указанный линк 10 раз
-vote [время в сек] [тема] [варианты] - запускает голосование на указанное время
    ОГРАНИЧЕНИЯ: одновременно только одно голосование
                 разделитель для вариантов выбора - пробел, поэтому, если вариант состоит из нескольких слов,
то нужно использовать нижнее подчёркивание
====================================
-info - группа команд с информацией, следом нужно указать команду:
    channelid - id канала
    guild - инфорация о сервере + список участников
    member [линк] - информация об участнике по линку
====================================
-mod - группа модерских комманд
===ограничение: только модераторы===
    moot [время сек.] [линк] - выдаёт плашку на временный мут в текстовых чатах
    voice [время сек.] [линк] - выдаёт плашку на временный мут в голосовых чатах
    banfunc [время сек.] [линк] - выдаёт плашку на временный бан функций бота
    clearmes [кол-во сообщений] [-exp] [линк] - удаляет последние сообщения
        [кол-во] - по уолчанию 10
        [-exp] - снять опыт за эти сообщения
        [линк] - с этим параметром удаляются сообщения только этого пользователя
====================================
-set - группа для изменения натроек
===ограничение: только модераторы===
    moot [линк/название] - меняет плашку для мута в тексовых чатах
    voicemoot [линк/название] - меняет плашку для мута в голосовых каналах
    banfunc [линк/название] - меняет плашку для бана функций бота
    recalc_stats - перерасчитывает статистику всех участников сервера, кроме времени в гс
====================================
-sys - группа системных комманд
===ограничение: только модераторы===
    write - записать текущую татистику
    off - запись статистики и отключение бота
